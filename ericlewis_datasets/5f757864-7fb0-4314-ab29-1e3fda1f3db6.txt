4. Output: 
CPU: Central Processing Unit, responsible for executing instructions and managing the computer's resources.
Memory: Stores data temporarily while it's being processed, typically in RAM (Random Access Memory).
Input Devices: Allow users to interact with the computer, like a keyboard, mouse, or touchscreen.
Output Devices: Display the results of processing, like a monitor or printer.
Storage Devices: Permanently store data, like a hard drive, SSD, or USB flash drive.
